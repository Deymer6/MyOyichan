# VozSalud 🎙️❤️

Aplicación móvil Android desarrollada para adultos mayores con enfermedades 
crónicas, que permite controlar la app completamente mediante comandos de voz 
o botones de alto contraste.

## Características principales
- 🎤 Control por voz con SpeechRecognizer (español peruano)
- 💊 Gestión de medicamentos con alarmas y recordatorios
- ❤️ Registro de signos vitales (presión arterial y glucosa)
- 🚨 Botón de emergencia con llamada directa al contacto configurado
- 🔐 Autenticación biométrica (huella dactilar / reconocimiento facial)
- 📊 Historial de registros médicos

## Tecnologías
- Kotlin · XML Layouts · MVVM
- Room Database · ViewModel · LiveData
- SpeechRecognizer · TextToSpeech
- BiometricPrompt · AlarmManager
- Navigation Component · Material Design 3
